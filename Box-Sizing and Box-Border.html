Box-Sizing and Box-Border

When using box-sizing you must use a webkit so it works in a range of browsers.

/* apply a natural box layout model to all elements */ *, *:before, *:after { -moz-box-sizing: border-box; -webkit-box-sizing: border-box; box-sizing: border-box;

The border and padding values moved inward, cutting into the width/height of the box rather than expanding it. Here is a dramatic example of the difference: (visual below) You have to think about the pixel count on the width, padding and border.
