Heidi's Take on HTML Semantics

HTML semantics help organize the content of websites/pages. These semantic elements create structure for your document, structure that helps both the reader/coder, and various processies/ors make sense of the content on your website/page. In other words semantic elements make it possible to create consistent outlines that are easy for search engines, web browsers & the visually impaired make sense of. HTML semantics label the page/website, making it clear what section represent what.
