DOM Nodes

DOM, an acronym short for document object model. DOMs are the APIs for altering the structure of HTML and XML. Nodes are the elements that make up the document, the family tree consisting of parents, children and siblings. Every HTML doc is a DOM node.

  • API= application program interface
  • XML= extensible markup language
  • DOM= document object model
