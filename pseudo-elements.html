Pseudo-Elements

Pseudo is a greek word and tranlates to mean, lying, deceitful or false. Pseudo elements allow you to add design elements to the CSS without marking up or changing the HTML. CSS-Tricks notes that “Pseudo elements are visible elements on a web page that aren't “in the DOM” or created from HTML, but are instead inserted directly from CSS. This allows you to do lots of neat design-y things without cluttering the markup.”

Pseudo classes vs. pseudo elements. Pseudo classes call directly on an element or node in the HTML to change. Pseudo elements on the other hand add a new element to the page. Side note :before and :after pseudo elements are really popular.
